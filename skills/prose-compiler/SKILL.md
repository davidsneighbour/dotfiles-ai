# Skill: Prose compiler

This skill converts draft text into concise technical prose.

It runs the following passes sequentially:

1. Structure pass
2. Clarity pass
3. Trope lint
4. Compression pass

# Workflow

Draft text enters the pipeline.

Pass 1: structure  
Reorganise the argument flow.

Pass 2: clarity  
Simplify vocabulary and sentence construction.

Pass 3: trope lint  
Remove AI rhetorical patterns.

Pass 4: compression  
Increase information density and remove repetition.

# Output characteristics

The final text should be:

concise  
precise  
mechanistic  
readable

The writing should resemble:

an engineer explaining a system to another engineer.

# When to use

Use this skill when:

- converting AI drafts into blog posts
- editing documentation
- cleaning verbose writing
- improving readability of long explanations
