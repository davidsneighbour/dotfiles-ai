# Clarity pass

Rewrite sentences to maximise clarity.

Focus on vocabulary and sentence construction.

# Simplify vocabulary

Replace inflated verbs:

utilize -> use  
leverage -> use  
delve -> explain  
facilitate -> enable

Replace abstract nouns when possible.

Example:

implementation of the optimisation strategy

becomes:

optimising the system

# Remove filler phrases

Delete phrases that add no information.

Examples:

it is important to note  
it is worth mentioning  
in many ways  
at the end of the day

# Prefer active voice

Rewrite passive constructions when possible.

Example:

The system was designed to reduce latency.

becomes:

The system reduces latency.

# Remove decorative adjectives

Avoid vague intensifiers:

significant  
powerful  
innovative  
remarkable

Replace with measurable claims.

Example:

significant performance improvement

becomes:

reduces latency by 40%
